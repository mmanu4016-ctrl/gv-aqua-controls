Key Features

Built for Industrial Precision

Engineered to deliver commercial-grade dependability inside Indian households.

Automatic Motor ON/OFF

Completely automates pump routines. Refills the overhead tank before it runs completely dry, and cuts power the second the tank fills.

Dry Run Protection

Constantly monitors pump load. If water supply at the sump source ceases, the controller shuts the motor off within 15 seconds to prevent overheating.

Under/Over Voltage Protection

Guards motor coils from grid fluctuations. Cuts supply if line voltage drops below 160V or surges above 260V, reconnecting automatically once safe limits return.

Intelligent Delay Timer

Prevents destructive short-cycling caused by temporary sensor waves. Adds a built-in buffer delay before re-enabling power switches.

Manual Override Switch

Dedicated industrial-grade bypass toggle enables manual overrides instantly if direct operational control is ever desired.
